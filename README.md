# Controls
- Move the camera [W S A D SPACE LSHIFT]
- Rotate OZ [Q E]
- Rotate OX [UP DOWN]
- ROTATE OY [LEFT RIGHT]
- Zoom in [C]
- Zoom out [V]
- Mode: [1 2 3]
  1 - edges only
  2 - painted walls
  3 - painted walls using width 1 (optimization demonstration)
